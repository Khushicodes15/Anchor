Anchor 

AI-powered mental wellness platform frontend

Anchor is a web-based mental wellness platform focused on narrative therapy, crisis readiness, and emotional resilience.
This repository contains the frontend for Anchor, built with a strong emphasis on accessibility, empathy, and animated storytelling.

🌍 Live Scope (Frontend)
Pages Implemented / Planned

Landing Page

Signup Page

Login Page

Dashboard (Post-login Home)

Narrative Journal

Safety Plan Builder

Crisis Mode (“I need help now”)

Anchor Wrapped (Monthly insights)

Community Library (Hope Archive)

Settings

🧠 Core Concepts

Narrative Therapy–inspired journaling

Crisis-ready design (human-first, non-alarming)

Personalized UI themes (“narrative worlds”)

Animated insight summaries (Spotify Wrapped–style)

Privacy-first mental wellness UX

🛠 Tech Stack

Next.js (App Router)

React

TypeScript

Tailwind CSS

Framer Motion (animations)

Lucide Icons

Backend and AI services are handled in a separate repository.

🚀 Getting Started

Clone the repository:

git clone https://github.com/<your-username>/anchor-web.git
cd anchor-web


Install dependencies:

npm install


Run the development server:

npm run dev


Open in browser:

http://localhost:3000
